Class GlobalDataObject

Base class for objects that are tracked in a global data store.

The data store is structured as follows:

ClassGroups = { class_group_1 = { 0 = 1 = ... n = }, class_group_2 = { ... },

...

class_group_n = { 0 = 1 = ... m = }, }

ClassGroupsSize = { class_group_1 = x class_group_2 = y ... class_group_n = z }
