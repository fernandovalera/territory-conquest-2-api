Class GameObject

Base class for data objects that have a CFrame and publish changes to their data to GlobalGameDataChanges.

All GameObjects can be listened to for data changes. Events are broadcast by subclasses in their setter methods.

Instances of this class behaves differently depending on whether they are running on the server or client. Only instances on the server will publish to GlobalGameDatachanges.
