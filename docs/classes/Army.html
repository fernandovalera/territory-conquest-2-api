Class Army

Represents of group of units of any type and Generals.

Units: units: The composition of units present in this Army. units_order: Tracks the number of unique unit types in this Army and provides the order of units units_class_map: Tracks the class maps of which there are at least 1 unit type of

Army Details: trained_by_structure_id: A marker for whether this Army is currently being trained. Logic can be built around whether this this army can move if it is being trained.

desired_units: The desired composition of units. Can allow progress to be shown or for Armies to be replenished by these.

desired_units_order: Tracks the number of unique types in this Army and provides the order of desired units
